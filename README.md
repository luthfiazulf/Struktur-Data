# Struktur-Data
Struktur Data adalah "sebuah cara mengorganisasi, menyimpan dan mengolah data agar dapat digunakan secara efisien oleh algoritma" 

# Tugas 1
Tugas 1 adalah selection sort yaitu algoritma sederhana dan intuitif untuk menyeleksi / mengurutkan satu elemen dengan nilai minimum dari sisa data yang ada sebelum melakukan pemindahan.
